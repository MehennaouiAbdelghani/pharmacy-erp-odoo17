# Pharmacy Reports

**Author:** Abdelghani Mehennaoui  
**Version:** 17.0.1.0.0

## Features

### Available Reports

1. **Expired Drugs Loss Report**
   - Lists all lots that expired in selected period
   - Calculates total financial loss
   - Shows product, lot, expiry date, and loss value

2. **Profit per Product Report**
   - Profit analysis by product
   - Revenue, cost, and profit breakdown
   - Profit margin percentage
   - Sortable by profitability

3. **Stock Valuation**
   - Integration with standard Odoo stock valuation
   - Filtered for selected date range

## Installation

Requires: `pharmacy_core`, `pharmacy_sales`, `pharmacy_purchase`

Install as #7 in sequence (final module)

## Usage

**Pharmacy > Reports > Generate Reports**

1. Select report type
2. Choose date range
3. Click "Generate Report"
4. PDF report will be generated
